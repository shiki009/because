Tab picker extension

Install the extension to pick from your open tabs. No cost, no Chrome Web Store — you load it yourself.

  1. Get the extension

    Download the Because project (e.g. from GitHub: Code → Download ZIP). The extension is in the extension folder.

  2. Open Chrome Extensions

    Go to chrome://extensions or Menu → Extensions → Manage extensions.

  3. Enable Developer mode

    Turn on the Developer mode toggle (top-right).

  4. Load unpacked

    Click Load unpacked, then select the extension folder from the extracted project.

  5. Use on your deployed app?

    By default the extension runs on localhost and file://. To use it on your deployed Because URL, edit extension/manifest.json and add your domain to the matches array. Example:

    // In manifest.json, content_scripts.matches:
"https://your-app.vercel.app/*"

A tab icon button will appear next to the content input when you use Because on a matching URL.
