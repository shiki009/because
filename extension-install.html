Tab picker

Pick from your open tabs when saving. Free — no Chrome Web Store needed.

Download extension (zip)

  1. Download

    Click the button above to get extension.zip. Unzip it — you'll get a folder with the extension files.

  2. Open Chrome Extensions

    Go to chrome://extensions or Menu → Extensions → Manage extensions.

  3. Enable Developer mode

    Turn on the Developer mode toggle in the top-right corner.

  4. Load unpacked

    Click Load unpacked and select the folder you extracted from the zip.

  5. Using on a custom domain?

    The extension runs on localhost and because-five.vercel.app by default. To add another domain, edit manifest.json in the extracted folder and add your URL to the matches array:

    "https://your-app.vercel.app/*"

Once installed, a tab-picker button appears next to the content input on Because.
