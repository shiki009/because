Tab picker extension

Install the extension to pick from your open tabs. No cost, no Chrome Web Store — you load it yourself.

Download extension (zip)

  1. Get the extension

    Click the button above to download extension.zip. Unzip it somewhere — you'll get a folder with the extension files.

  2. Open Chrome Extensions

    Go to chrome://extensions or Menu → Extensions → Manage extensions.

  3. Enable Developer mode

    Turn on the Developer mode toggle (top-right).

  4. Load unpacked

    Click Load unpacked, then select the folder you extracted from the zip.

  5. Use on your deployed app?

    By default the extension runs on localhost and file://. To use it on your deployed Because URL, edit manifest.json in the extracted folder and add your domain to the matches array. Example:

    // In manifest.json, content_scripts.matches:
"https://your-app.vercel.app/*"

A tab icon button will appear next to the content input when you use Because on a matching URL.
